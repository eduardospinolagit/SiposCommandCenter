Backend

Edge Functions — o motor da integração

✨ gerar-script
Chamada pelo SLAC ao clicar "Gerar com IA" — pesquisa o lead e retorna script personalizado
  1. 1Recebe: instagram, negocio, cidade do lead
  2. 2Apify busca perfil Instagram (bio, posts, seguidores)
  3. 3Apify busca Google Maps (avaliações, categoria, site)
  4. 4Claude API monta contexto + adapta script base
  5. 5Retorna script pronto para revisar e enviar
📨 wa-webhook
Chamada pela Z-API quando o lead responde — persiste a mensagem e notifica o SLAC
  1. 1Z-API envia POST com mensagem recebida
  2. 2Valida token secreto (segurança)
  3. 3Busca lead pelo número de telefone
  4. 4Insere na tabela conversas
  5. 5Supabase Realtime notifica o SLAC instantaneamente
